# ESD Paris Digital Event 2023 - Exp Visiteur - API

API for 2023 ESD Paris Digital Event reservations

## Environment variables

```
API_PORT=
AIRTABLE_TOKEN=
AIRTABLE_BASE=
```

## Usage

Make sure to add env variables in a .env at the project's root.

### Setup

```
npm install
```

### Dev

```
npm run dev
```

### Production

```
npm run start
```
